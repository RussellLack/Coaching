# Executive OS Claude Design Handoff

This folder contains a working HTML/CSS/JS prototype for a premium coaching service for experienced professionals navigating AI disruption.

## Files

- `index.html` - customer-facing website and assessment journey
- `review.html` - private reviewer workspace for market testing
- `styles.css` - shared visual system and responsive styling
- `app.js` - live assessment logic
- `review.js` - private review feedback form logic
- `assets/ai-navigation-visual.png` - hero visual asset
- `claude-design-brief.md` - prompt and UX brief for Claude Design
- `customer-journey-map.md` - desired journey structure and improvement priorities

## Suggested Claude Design Use

Upload this folder, or upload the ZIP version if available. Ask Claude Design to refine the UX, interaction design, visual hierarchy and customer journey smoothness while preserving the strategic positioning and tone of voice.

The key instruction is to improve the experience, not to turn it into a generic coaching landing page.

The current separation is intentional:

- `index.html` should feel like the real public site.
- `review.html` should feel like a private testing workspace.
